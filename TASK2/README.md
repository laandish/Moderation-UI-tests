Требования

Перед запуском должны быть установлены:

Go
git

Установка зависимостей

Перейти в директорию проекта:

cd TASK2

Установить зависимости модуля:

go mod tidy
Настройка
 Проверить go.mod

В корне проекта должен быть файл go.mod.

Команда запуска всех тестов
go test -v ./...
Запуск только тестов по объявлениям
go test -v ./tests/scenarios/advertisements/...
Запуск одного теста

Пример:

go test -v -run TestCreateItemPositive ./...
Примеры покрываемых сценариев

Автотесты покрывают:

создание объявления
получение объявления по id
получение списка объявлений по sellerID
получение статистики
удаление объявления
негативные сценарии валидации
E2E сценарии
проверки идемпотентности
часть нефункциональных проверок

В процессе тестирования были обнаружены дефекты API. Они описаны в файле:

BUGS.md

Часть сценариев, связанных с найденными дефектами, может падать до исправления сервиса.


Обновить зависимости:

go mod tidy

Форматировать код:

go fmt ./...

Проверить список пакетов:

go list ./...
Результат запуска

При успешном запуске в консоли отображаются статусы тестов:

PASS - тест пройден
FAIL - тест упал
