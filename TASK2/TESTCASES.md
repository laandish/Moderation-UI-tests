
## Общие условия для всех тестов

- **Окружение:** платформа Web API, операционная система macOS, браузер Chrome
- **Base URL:** `https://qa-internship.avito.com`
- **Формат обмена:** `application/json`
- **Примечание:** для изоляции тестов использовать уникальный `sellerID` в диапазоне `111111–999999`

## **ID: TC-001**

**Заголовок:** Успешное создание объявления с валидными данными

**Предусловия:**  
Сервис доступен, подготовлен уникальный `sellerID`

**Шаги:**
1. Отправить `POST /api/1/item` с валидным JSON-телом
2. Проверить тело ответа

**Ожидаемый результат (ОР):**
- Статус ответа `200`
- В ответе присутствуют поля `id`, `sellerId`, `name`, `price`, `statistics`, `createdAt`
- Значения `sellerId`, `name`, `price`, `statistics` соответствуют отправленным
- `id` не пустой

**Статус:** Passed / Failed / Skipped / Blocked

**Постусловия:**  
Удалить созданное объявление через `DELETE /api/2/item/{id}`

**Окружение:**  
платформа Web API, операционная система macOS, браузер Chrome
