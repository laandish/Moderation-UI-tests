# BUGS.md

## BUG-001

**Краткое описание:**  
API позволяет создать объявление с отрицательной ценой

**Шаги воспроизведения:**
1. Отправить `POST /api/1/item`
2. Передать валидное тело запроса, но указать отрицательное значение `price`, например:

```json
{
  "sellerID": 123456,
  "name": "negative price item",
  "price": -100,
  "statistics": {
    "likes": 1,
    "viewCount": 10,
    "contacts": 1
  }
}
Проверить статус ответа

Фактический результат:
Сервис возвращает 200 OK и создает объявление

{
  "status": "Сохранили объявление - <id>"
}

Ожидаемый результат:
Сервис должен вернуть 400 Bad Request, так как цена не может быть отрицательной

Серьёзность:
Medium

Окружение:
Платформа: Web API
ОС: macOS
Браузер: Chrome
Base URL: https://qa-internship.avito.com
