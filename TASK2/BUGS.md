
## BUG-001: API позволяет создавать объявление с отрицательной ценой

**Краткое описание:**
Сервис принимает отрицательное значение `price` и успешно создает объявление.

**Шаги воспроизведения:**

1. Отправить POST `/api/1/item`
2. Передать `price: -100` в теле запроса

**Фактический результат:**

* Статус: 200 OK
* Объявление успешно создается

**Ожидаемый результат:**

* Статус: 400 Bad Request
* Ошибка валидации (price не может быть отрицательным)

**Серьёзность:** High

**Окружение:**
Web API, macOS, Chrome



## BUG-002: API позволяет создавать объявление с отрицательными значениями статистики

**Краткое описание:**
Сервис принимает отрицательные значения `likes`, `viewCount`, `contacts`.

**Шаги воспроизведения:**

1. Отправить POST `/api/1/item`
2. Передать:

   ```json
   "statistics": {
     "likes": -1,
     "viewCount": -10,
     "contacts": -5
   }
   ```

**Фактический результат:**

* Статус: 200 OK
* Объявление создается

**Ожидаемый результат:**

* Статус: 400 Bad Request
* Ошибка валидации

**Серьёзность:** High

**Окружение:**
Web API, macOS, Chrome



## BUG-003: Несоответствие контракта — POST /api/1/item не возвращает id в явном виде

**Краткое описание:**
В ответе на создание объявления отсутствует поле `id`, вместо него возвращается строка в поле `status`.

**Шаги воспроизведения:**

1. Отправить POST `/api/1/item`
2. Проверить тело ответа

**Фактический результат:**

```json
{
  "status": "Сохранили объявление - <uuid>"
}
```

**Ожидаемый результат:**

```json
{
  "id": "<uuid>",
  "sellerId": ...,
  "name": ...,
  "price": ...,
  "statistics": {...},
  "createdAt": ...
}
```

**Серьёзность:** Medium

**Окружение:**
Web API, macOS, Chrome



## BUG-004: API принимает отрицательный sellerID и возвращает 200 вместо ошибки

**Краткое описание:**
Сервис принимает отрицательный `sellerID` в path и возвращает успешный ответ.

**Шаги воспроизведения:**

1. Отправить GET `/api/1/-1/item`

**Фактический результат:**

* Статус: 200 OK
* Возвращается корректный ответ

**Ожидаемый результат:**

* Статус: 400 Bad Request или 404
* Ошибка валидации

**Серьёзность:** Medium

**Окружение:**
Web API, macOS, Chrome
